# AppKloser

#¡Conecta con tus amigos de manera simple y divertida! Kloser te permite registrarte, iniciar sesión y mantener tu lista de amigos siempre a la mano. Además, puedes capturar momentos con la cámara directamente desde la app y almacenar tus fotos junto a tu perfil. Todo en una interfaz moderna y fácil de usar, desarrollada con Jetpack Compose.
